# Helden Inc. — Homepage Showcase

A sleek, gamified marketing homepage for **Helden Inc.** — pioneers in gamified internal communication. Built with Streamlit for instant deployment.

## 🚀 Deploy to Streamlit Cloud

1. Fork or clone this repo to your GitHub account
2. Go to [share.streamlit.io](https://share.streamlit.io)
3. Click **New app** → select this repo
4. Set **Main file path** to `helden_app.py`
5. Click **Deploy** — done ✅

## 🏃 Run locally

```bash
pip install -r requirements.txt
streamlit run helden_app.py
```

## 📁 File structure

```
├── helden_app.py       # Main Streamlit app (single file)
├── requirements.txt    # Dependencies
└── README.md
```
