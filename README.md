# Helden Inc. — Cinematic Experience

12-slide Orano-style fullscreen showcase of helden-inc.com/en/#games

## Deploy to Streamlit Cloud (free, 3 steps)

1. Create a **public GitHub repo** and upload:
   - `helden_slider.py`
   - `requirements.txt`
2. Go to **share.streamlit.io** → New app
3. Set **Main file**: `helden_slider.py` → Deploy ✅

## Design System

- **Fonts**: Cormorant Garamond (display) · Outfit (body) · DM Mono (labels)
- **Colors**: #07070a void-black + #FFD600 Helden yellow
- **Transitions**: Film-shutter cut (70ms blackout) + scale + blur cinema system
- **Animations**: Up-float, fade, clip-reveal (horizontal wipe), scale-in — each with independent stagger timing
